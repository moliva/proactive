A fully peer-to-peer chat application using Group communication and migration. Each new client asks an other connected client for the list of users taking part of the chat. After that, all communications are provided by the ProActive Group Communication mechanism. This aplication has the ability to migrate. The user is able to continue a chat intialized on a desktop with a Laptop or a PDA wireless-connected.
