Provides the ProActive Network Protocol, PNP

PNP is a communication protocol for ProActive. It's purpose is to replace RMI as the default communication protocol. Indeed RMI has some severe limitations for our use:

  • Not firewall/NAT friendly (use a lot unpredictable TCP port)
  • Does not support timeout
  • Not scalable due to massive thread number

PNP tries to address all theses limitations will being simple, lightweight and efficient
