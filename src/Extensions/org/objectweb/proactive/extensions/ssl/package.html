Provides base classes and helper to use SSL in ProActive. Theses classes are intended to be reused by SSL enabled protocol. They should all use the same setup, protocol, ciphered suite, configuration etc.

This package provides class to:

  • Generate keypair, certificate and keystore
  • Read, write and update keystore stored on hard drive
  • Preconfigured trust managers
  • Preconfigured SSL enabled socket factory (client and server)
  • Preconfigured SSL engine (client and server)

@author ProActive Team @since ProActive 4.4.0
